¿Qué es CSS?

CSS, Cascading Style Sheets, Hoja de estilo en cascada que describre la presentación de los documentos estructurados en hojas de estilo para diferentes métodos de interpretación.

¿Para qué sirve?

CSS fue desarrollado por el W3C (World Wide Web Consortium) con el fin de separar los contenidos de los documentos, HTML, XML, XHTML, SVG o XUL de la presentación del documento con las hojas de estilo, incluyendo elementos tales como: Colores, fondos, márgenes, bordes, tipos de letra, tamaños, ETC, permitiendo controlar el formato y estilo de los documentos.
¿Cómo funciona?

El Lenguaje CSS Se basa en una serie de reglas que rigen el estilo de los elementos en los documentos estructurados. Cada regla consiste basicamente en un selector y una declaración. La declaración va en entre corchetes y consiste en una propiedad o atributo y un valor separados por dos puntos.



Por ejemplo: h1 {font-size:50px}
h1 --> Es el selector
{font-size:50px} --> Es la declaración
font-size --> Es la propiedad o atributo
50px --> Es el valor de la propiedad


La información CSS se puede proporcionar por varias fuentes, ya sea incorporado o adjunto en el documento HTML. Destacan tres formas de dar un estilo, las cuales son:

1: Hoja de estilo en línea: sirven para aplicar las propiedades de las hojas de estilos en cascada a elementos individuales de una página sin tener que reutilizar el estilo. Los estilos en línea se definen dentro de las etiquetas iniciales de los elementos HTML de páginas web.

2. Hoja de estilo interna: irven para aplicar estilos a una sola web y también para modificar los estilos de heredados de una CSS externa. Las CSS internas se incluyen entre las etiquetas de las páginas web.

3. Hoja de estilo externa: La Hoja de Estilo Externa se almacena en un archivo diferente al del archivo con el código HTML al estar vinculado a través del elemento link
